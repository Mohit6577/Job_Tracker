# Job Tracker App

A full-stack Job Tracking application where users can add, delete, search, and filter job applications.

## 🚀 Features

* Add job (company, role, status, notes)
* Delete job
* Search jobs
* Filter by status (Applied / Interview / Rejected)

## 🛠 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js + Express
* Database: MongoDB

## 📦 Setup

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev

## 🌐 API Endpoints

* GET /api/jobs
* POST /api/jobs
* DELETE /api/jobs/:id
